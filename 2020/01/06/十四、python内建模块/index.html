十四、Python内建模块

Python中大多数系统级接口都集中在sys和os这两个模块，这样说过于简单化了，还有一些其他的标准模块也属于这个领域，他们包括：

glob 用于文件名扩展
socket 用于网络连接和进程间通信（IPC）
threading，thread，queue 用于运行和同步化并发线程
time，timeit 用于获取系统时间相关细节
subprocess，multiprocessing 用于启动和控制并行进程
signal，select，shutil，tempfile 用于多种系统相关任务

一、sys和os

原则上来说，sys负责导出与Python解释器本身相关的组件（如模块搜索路径），os则包含与Python所在底层操作系统相对应的变量和函数。

os._exit()会直接将python程序终止，之后的所有代码都不会继续执行。没有异常

sys.exit()会引发一个异常：SystemExit，如果这个异常没有被捕获，那么python解释器将会退出。如果有捕获此异常的代码，那么这些代码还是会执行。
