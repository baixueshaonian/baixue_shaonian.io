三、创建模块

定义模块，只要使用文本编辑器，把一些Python代码输入至文本文件中，然后以‘.py’为后缀名进行保存，任何此类文件都会被自动认为是Python模块。在模块顶层指定的所有变量名都会变成其属性（与模块对象结合的变量名），并且可以导出供客户端来使用

1、创建第一模块：

#！/usr/bin/env python3

在linux环境下，用来指定脚本解释器

​ 最简单的方法是which python[3],找到路径但是这是硬编码不推荐。

​ 推荐使用#！/usr/bin/env python[3]这样会自动搜索脚本解释器的绝对路径。

在硬编码中如果拷到其他机器呢么，硬编码可能就失效了。

通过#!的方法会自动寻找python解释器的位置。
