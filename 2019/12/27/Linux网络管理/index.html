Linux网络管理

  • RedhatLinux
  • 2019-12-27

简介：详细描述了如何对RedLinux系统进行网络管理

一、查看当前的网络的基本信息：

1、通过ifconfig命令查看网卡信息：

ifconfig eth0查看指定网卡的信息：

通过ip addr命令查看网卡信息（不是所有系统都可用）

2、通过ip -s link show eno16777736查看网卡流量信息

3、通过route -n查看当前路由表

上图中可看到一条默认路由指向192.168.199.2 ，因为我使用的是虚拟机所以该ip地址就是对应我的本地物理机上的虚拟网卡的ip地址，也就是虚拟机的网关。

一条是指向192.168.199.0网段（本网段）的泛洪地址。

4、网络测试：ping

linux与windows的ping区别是无限ping

ping -c【n】+ IP地址 （通过n来限制其次数）

5、显示本机到网关到底经过多少跳：

traceroute www.baidu.com

6、netstat -anpt 查看当前所有的连接端口

二、使用nmcli命令配置网络：

使用nmcli命令和NetworkManager管理网络设置和设备

NetworkManager是一个用来监视和管理网络设置的守护进程

nmcli是一个用来配置网络设置的命令行工具。

nmtui提供图形化接口来配置网络。

还有个GNOME小程序可以显示网络状态和设置。

1、显示所有连接

nmcil connection show

UUID设备的唯一标识

2、查看设备状态：

nmcil device status

三、编辑网卡的配置文件：

1、网卡配置文件

cd /etc/sysconfig/network-scripts/

systemctl restart network 重启网络

查看现象：

2.dns配置文件

cat /etc/resolv.conf

查看当前的dns服务器。

3、本地的域名解析

cat /etc/hosts

4、修改当前的主机名：

cat etc/sysconfig/network

hostname 直接进行修改。即时的要想永久生效，必须修改配置文件。
